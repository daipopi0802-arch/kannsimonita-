# 🛠️ CORE_INTEL: Cyberpunk News Widget

Windowsデスクトップに常駐する、近未来的なデザインの多機能ニュースウィジェットです。  
最新のニュース、市場データ、システムの状態を、サイバーパンクなインターフェースで一元管理できます。

---

## ✨ 主な機能 (Key Features)

* **📡 リアルタイムRSSフィード**: ReutersやBloombergなどの最新ニュースを自動取得し、リスト表示します。
* **💹 ライブ・マーケットティッカー**: 日経平均、DOW、NASDAQ、ドル円などの主要指標を最下部にスクロール表示。
* **📊 システムモニタリング**: CPUとRAMの使用率をスタイリッシュなバーグラフでリアルタイムに監視。
* **🎨 カスタムテーマ**: 3つのプリセット（CYBER_CYAN, MATRIX_GREEN, NEON_PINK）を即座に切り替え可能です。
* **☁️ 天気 & 時計**: 現在の天気、気温、正確な時刻をウィジェット内に統合表示します。
* **📝 デスクトップ・メモ**: 忘れてはいけないタスクやリマインダーを直接書き込み、保存できます。
* **🎬 シネマモード**: ニュースフィードを自動的にスクロールさせるハンズフリー閲覧モードを搭載。
* **⌨️ グローバルホットキー**: `Win + Shift + N` で、どの画面からでも瞬時に表示/非表示を切り替えられます。
